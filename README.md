# homebrew-grove

Homebrew tap for [grove](https://github.com/Entelligentsia/grove) — structural,
byte-precise, token-cheap codebase access for coding agents.

```sh
brew install Entelligentsia/grove/grove
```

The formula installs a prebuilt binary from grove's GitHub Releases (macOS and
Linux, x86_64 + aarch64). It is regenerated on each release with
`dist/homebrew/update-formula.sh <tag>` in the grove repo.
